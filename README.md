# UrbanPlanner

UrbanPlanner is a 3D real estate exploration platform that allows users to discover construction projects through interactive 3D models and AR.

The platform enables builders to upload and manage projects, while users can explore properties visually and interactively.

## Features

### User Features

- Browse real estate projects
- View project details
- Interactive 3D model viewer
- Image gallery
- Project filtering and search

### Builder Features

- Builder authentication
- Upload new projects
- Manage existing projects
- Edit project details
- Delete projects
- Builder dashboard for project management
